分析

1.先要解决pattern的问题即
if(preg_match($pattern,$str,$arr))
str=pattern
这个很简单,把代码放在本地,然后echo $pattern;即可
2.解决num的判断
is_numeric($num) and $num!=='36' and trim($num)!=='36' and filter($num)=='36'
is_numeric($num) 判断是否数字或者数字字符串
trim($num) 去掉首尾空格
filter($num) 这个函数是自己写的,作用是把某些特定字符转变为1
解决方案: %0c36
is_numeric常见绕过: %0c , 十六进制绕过 , %00绕过(前后都可以) , %20(只能放在数字后)
由于有trim()因此不能用%20,有filter()不能用%00和十六进制
因此只能用36a有不符合
#为什么%0c可以而%0a,%0b,%0d不行????
#%0c会被解析成存在的特殊字符,如下图中的一个箭头
#%0a,%20会被解析成空格,过不了trim($num)!=='36'
#%0c只能放前面

4.phar协议结合反序列化

源码分析:

class.php
<?php
class upload{
    public $filename;
    public $ext;
    public $size;
    public $Valid_ext;

    public function __construct(){
        $this->filename = $_FILES["file"]["name"];
        $this->ext = end(explode(".", $_FILES["file"]["name"]));
        $this->size = $_FILES["file"]["size"] / 1024;
        $this->Valid_ext = array("gif", "jpeg", "jpg", "png");
    }

    public function start(){
        return $this->check();
    }

    private function check(){
        if(file_exists($this->filename)){
            return "Image already exsists";
        }elseif(!in_array($this->ext, $this->Valid_ext)){
            return "Only Image Can Be Uploaded";
        }else{
            return $this->move();
        }
    }

    private function move(){
        move_uploaded_file($_FILES["file"]["tmp_name"], "upload/".$this->filename);
        return "Upload succsess!";
    }

    public function __wakeup(){
        echo file_get_contents($this->filename);
    }
}


class check_img{
    public $img_name;
    public function __construct(){
        $this->img_name = $_GET['img_name'];
    }

    public function img_check(){
        if(file_exists($this->img_name)){
            return "Image exsists";
        }else{
            return "Image not exsists";
        }
    }
}

关注到 check_img中的file_exists()中的参数是我们可以控制的(该参数和include()一样),因此考虑phar反序列化

注意：复现phar反序列化的时候要将php.ini中的phar.readonly选项设置为Off，否则无法生成phar文件。
#流程: 在本地写好php文件,利用这个php文件生成shell.phar文件,再将生成的shell.phar改名为shell.png
将这个shell.png上传后,利用phar://shell.png去读取该文件,触发反序列化
